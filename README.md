# Java GUI Calculator

A desktop calculator application built using Java Swing.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Decimal calculations
- Reset functionality
- GUI-based interface
- Undefined result when dividing by zero

## Technologies Used

- Java
- Java Swing
- Eclipse IDE
